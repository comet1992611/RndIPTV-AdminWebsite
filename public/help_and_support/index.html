Hello from Magoware

If you are having troubles using our application, please read the full instructions below:


    The Login page allows you to input your username and password and open your account.

    If you don't have already an account, you can Sign Up at the SignUp page which is located in the right side of Login Page, you can access it by scrolling right while standing at Login Page. As you get to the Sign Up Page, fill the fields with your information and Submit it.

    The information you will submit will be used to log you in your account.

    After logging in, you will find yourself in the Main Menu.

    The first icon is named LiveTV. It is used to play the streams that you will add. To add a new stream, you should go to Personal menu. This Menu allows you to modify your data and information and at the last section named "My Channels", you can Create, Edit or Remove a channel. If you haven't added a channel yet, create one with "Create" button adding the channel's title, description and stream source. After submitting these changes, go back to Main Menu and play the added channel at "Live TV" category.

    Live TV offers multiple services. You can navigate through your channels by swiping left and right in case you have multiple channels added. In the right, you can find four buttons. Using the first one, you can play and pause the current channel. In the second button, you can get information about the current channel's program. In the third one you are able to mark a channel as Favorite. In the last one, you can get information about all the channel's program guide.

    If you have a large number of channels, you can access them more easily. By long pressing into the screen, you will get the full channels list in the bottom of screen which you can scroll easily to select the channel you were looking for.
